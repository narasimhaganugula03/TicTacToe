# Tic Tac Toe Game 🎮

A simple Tic Tac Toe game built using Python and Tkinter. This project provides an interactive graphical user interface (GUI) where two players can play Tic Tac Toe on a 3x3 board.

## Features

* Interactive GUI using Tkinter
* Two-player gameplay (X and O)
* Automatic win detection
* Draw detection
* Restart game functionality
* Simple and user-friendly interface

## Technologies Used

* Python 3
* Tkinter

## Installation

1. Clone the repository:

```bash
git clone https://github.com/narasimhaganugula03/TicTacToe.git
```

2. Navigate to the project directory:

```bash
cd TicTacToe
```

3. Run the application:

```bash
python tictactoe.py
```

## How to Play

1. Launch the game.
2. Player X starts first.
3. Players take turns clicking an empty cell.
4. The first player to align three symbols horizontally, vertically, or diagonally wins.
5. If all cells are filled and no player wins, the game ends in a draw.
6. Click "Restart Game" to play again.
